## Repository Purpose

GenAI-101 is an educational repository for teaching Generative AI concepts from scratch. It contains Jupyter notebooks, code examples, and learning materials that accompany video tutorials on YouTube (https://www.youtube.com/@kamaldhungana-g1j).

## Repository Structure (Evolving)

The repository is in early stages. Materials will be organized to align with video lessons, with notebooks and code examples added progressively as the course develops.

## Content Guidelines

When adding new materials:

- **Notebooks**: Should be self-contained with clear explanations, code cells, and outputs that match video lessons
- **Code examples**: Must be runnable independently and include comments for learners
- **Educational focus**: All content is designed for beginners learning GenAI - prioritize clarity over optimization
- **YouTube alignment**: New materials should reference corresponding video lessons where applicable

## Working with Notebooks

This repository will contain Jupyter notebooks (.ipynb files). When working with them:

- Use the NotebookEdit tool for modifications
- Ensure notebooks run top-to-bottom without errors
- Keep explanatory markdown cells clear and beginner-friendly
- Test code cells before committing
